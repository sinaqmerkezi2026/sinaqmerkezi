# **App Name**: ImtahanFlow

## Core Features:

- Secure Admin & Student Access: Implement distinct authentication flows: a hidden URL with a specific password for admin panel access, and unique 6-character codes plus student identity for exam access and results viewing.
- Comprehensive Exam Creation: Admins can create new exams, define properties (name, active start/end dates, duration), and automatically generate 100 unique alphanumeric access codes per exam.
- Versatile Question Authoring: Enable admins to add three types of questions: multiple-choice (with selectable correct answers), open-ended (with predefined correct answer for comparison), and explanation-based (requiring correct final answer and detailed explanation criteria, with image upload capability).
- AI-Powered Explanation Grader Tool: A generative AI tool that automatically compares student-provided explanations against admin-defined criteria for explanation-based questions, assigning scores (0, 1/3, 1/2, 2/3, 1) and providing contextual feedback on the reasoning.
- Student Exam Experience: Students access and take exams by entering a valid code, their name, and surname. The system manages the exam timer and presents questions sequentially.
- Automated Scoring & Detailed Results: After exam completion, students receive immediate results including overall score, correct/incorrect answer breakdown, and specific AI-generated feedback for explanation-based questions. Results are accessible via the original exam code.
- Exam Sales Integration: Admins can set a price for each exam, and students can initiate an inquiry to purchase exam codes via a pre-filled WhatsApp or SMS message to +994514262676.

## Style Guidelines:

- Color scheme: A light theme optimized for readability and focus, reflecting clarity and professionalism for an online examination system.
- Primary color: A stable and intelligent blue (#3A75B2). This choice instills trust and promotes focus, serving as a prominent yet balanced element within the light theme.
- Background color: A very light, desaturated blue (#F0F4F8). This hue provides a serene and clean canvas for displaying content, maintaining visual harmony with the primary color and aiding concentration.
- Accent color: A fresh and contrasting blue-cyan (#33BACC). This vibrant color will effectively highlight interactive elements, calls-to-action, and key information, ensuring clear visibility and encouraging user interaction.
- Main font: 'Inter' (sans-serif) for all text. This choice ensures excellent readability across various screen sizes due to its neutral, modern, and objective aesthetic, which is ideal for an efficient examination interface.
- Icons: Clean, simple, and highly intuitive vector-based icons. These will communicate their function clearly without causing distraction, thereby enhancing usability and simplifying navigation within the application.
- Layout: A minimalist and highly responsive design. The layout will prioritize content clarity and a smooth user progression, ensuring an optimal experience on both mobile and desktop devices. Strategic use of ample white space will guide the user's eye and minimize cognitive load during exams.
